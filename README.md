🚴 Jenson USA Sales Analysis | SQL Project

This project showcases my ability to use SQL for data cleaning, querying, aggregation, window functions, and business analysis. It demonstrates how raw data can be transformed into meaningful insights that support strategic business decisions.

## 📌 Project Overview
This project is a SQL-based sales and business analysis using the Jenson USA Bike Store dataset. The objective is to solve real-world business problems by writing optimized SQL queries and applying advanced SQL concepts.

## 🛠️ SQL Concepts Covered
- SELECT, WHERE, ORDER BY
- INNER JOIN, LEFT JOIN, USING
- GROUP BY and HAVING
- Aggregate Functions (SUM, COUNT, AVG, MAX, MIN)
- Window Functions (ROW_NUMBER, SUM() OVER())
- Subqueries and Nested Queries
- Aliases and Calculated Columns
- Data Filtering and Sorting

## 📊 Business Problems Solved
- Total products sold by each store.
- Cumulative quantity sold for each product over time.
- Highest-priced product in each category.
- Customer who spent the most money.
- Staff members who have not made any sales.
- Staff members with above-average sales performance.
- Customers who purchased products from every category.
- Sales and revenue analysis using discounts and total sales calculations.

## 🗂️ Database Tables Used
- Customers
- Orders
- Order Items
- Products
- Categories
- Stores
- Staffs
- Brands
- Stocks
