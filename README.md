# Jenson_USA_Sales_Advance_SQL_Project
A SQL data analytics project based on the Jenson USA Bike Store dataset, featuring advanced SQL queries, joins, window functions, subqueries, and business-driven sales analysis to solve real-world data problems. 🚴📊

🚴 Jenson USA Sales Analysis | SQL Project

This project is a comprehensive SQL-based sales and business analysis using the Jenson USA Bike Store dataset. The objective is to solve real-world business problems by writing optimized SQL queries and applying concepts such as joins, aggregations, window functions, subqueries, and common analytical techniques.

📌 Project Objectives
Analyze sales and order data across multiple stores.
Identify customer purchasing behavior and product trends.
Evaluate staff and store performance.
Generate business insights using advanced SQL queries.
🛠️ SQL Concepts Covered
SELECT, WHERE, ORDER BY
INNER JOIN, LEFT JOIN, USING
GROUP BY and HAVING
Aggregate functions (SUM, COUNT, AVG, MAX, MIN)
Window functions (ROW_NUMBER(), SUM() OVER())
Subqueries and nested queries
Aliases and calculated columns
Conditional filtering and ranking
📊 Business Questions Solved
Total products sold by each store.
Cumulative quantity sold over time for each product.
Highest-priced product in each category.
Customer who spent the most money.
Staff members who have not made any sales.
Staff members with above-average sales performance.
Customers who purchased products from every category.
Product sales analysis with discounts and total revenue calculations.
Store-wise and customer-wise order analysis.
